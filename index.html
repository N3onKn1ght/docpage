Introduction

What is HTML? HTML stands for "HyperText Markup Language" and is used to build what you are looking at right now, a webpage. It is used with another language called Cascading Style Sheets, or CSS. Both of these languages are relatively easy to learn, and they are fun too!

These languages have gone over a big range of development over the years, and it is ever changing. There have been so many suggested improvement or ideas that it could have gotten a little messy (and if not done correctly, it sometimes is). This is why a couple organizations have risen to set standards for web design, to keep things consistent for everyone.

Two of these organizations are:

  • World Wide Web Consortium
  • Web Hypertext Application Technology Working Group

HTML Basics

HTML is a really simple language. It consists of several pieces to make a whole page. These pieces are called elements. Elements have opening and closing "tags" that look like this:

<p> </p>

Notice that the last tag has a slash before the type of element. That specific element was a paragraph element. There are several elements that are used in a webpage. You can find a list of them here. There are a few other things that need to be covered. Every page must be started with a "declaration," like this one:

<!DOCTYPE html>

The above code helps browsers render the html. There are other types of html to specify, but this is the universal way with the newest updated HTML5 version. All code (besides the declaration, it is always on top) should also be wrapped in an  <html></html>  tag. There should be a head element and a body element in between the html tags, with head being on top (opening and closing tags just like the other elements). The head tag is for elements that help with Search Engine Optimization, like meta tags. It is also used for displaying the title in the browser tab and linking the stylesheet. Some elements are self closing, meaning you don't need to put an end tag like the other elements. Some of these are:

  • <img>
  • <meta>
  • <link>

All you have to do is use the elements exactly how they are shown, with only one tag. HTML uses attributes to enhance certain element or provide sources (For example, links or links to images) to information. For image, the attributes needed are:

<img src="image.png" alt="an image" height="200" width="300">

As you can see, the src attribute loads the image from the source file, the alt attribute helps with accessability (screen readers), and the height and width attributes help the page to load quicker by reserving space for the image. There are many attributes, and there is a list of them here.

CSS Basics

CSS, or Cascading Style Sheets is a way to enhance HTML's simple structure, by adding color, changing an elements size, or even animating an element. Theres a lot to learn about this language, and a full documentation can be viewed here. Below is an example of CSS being used to change the background color of the  <body>  element.

body { background-color: red; }

Different Ways to Use CSS

As stated in the above section, there are a few ways you can use CSS to enhance HTML.

Some of these ways are:

  • Changing element color
  • Changing page layout
  • Changing text font

Most of page design exists in the CSS, and it comes with very useful tools.

Other Tools

Another well known way of enhancing web pages is by using JavaScript. JavaScript can use functions to make changes to a page real time on the client side, whn a user interacts with it. There is also extensions for css. SASS, or Syntactically Awesome Stylesheets, is a powerful language extension the improves the use of CSS. There is also bootstrap, a language framework that helps design websites.
